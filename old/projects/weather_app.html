About the project

A project made with React for the front-end using RestAPI.

Click here to get to Github Repository

This is an application that shows the weather of any city. As a plus point, you can type the city name in any language and get the weather details.

Technologies Used
  • React
  • RestAPI

npm start

Runs the app in the development mode. Open http://localhost:3000 to view it in your browser. The page will reload when you make changes. You may also see any lint errors in the console.

npm run build

Builds the app for production to the build folder. It correctly bundles React in production mode and optimizes the build for the best performance. The build is minified and the filenames include the hashes. Your app is ready to be deployed!
